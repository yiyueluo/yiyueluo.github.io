Fabrication

Manual Fabrication

A resistive tactile sensing matrix can be manually fabricated by aligning top and bottom electrodes orthogonally, sandwiching resistive layers between them. All necessary materials are commercially available, and various conductive materials can be used for the electrodes to achieve different sensing performance characteristics.

Digtal Fabrication via Machine Embroidery

A resistive tactile sensing matrix can be digitally fabricated using machine embroidery. Conductive thread is stitched over a resistive layer as both the top and bobbin thread in separate passes, creating a similar sandwiching structure.
